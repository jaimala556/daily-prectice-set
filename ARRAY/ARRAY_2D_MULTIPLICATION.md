# Array 2D matrix Multiplication

**Problem**: Multiply two 2x2 matrices.

**Input**:

```
Matrix A = [[1, 2], [3, 4]]
Matrix B = [[5, 6], [7, 8]]
```

**Output**:

```
Result = [[19, 22], [43, 50]]
```

---

**Problem**: Multiply two 3x3 matrices.

**Input**:

```
Matrix A = [[1, 0, 2], [-1, 3, 1], [3, 2, 1]]
Matrix B = [[2, 3, 1], [0, 1, 0], [1, 2, 3]]
```

**Output**:

```
Result = [[4, 7, 7], [1, 5, 2], [8, 13, 6]]
```

Result = [[58, 64], [139, 154]]
