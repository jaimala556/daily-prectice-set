# Array 2D matrix Addition Subtraction

**Problem**: Add two 2x2 matrices.

**Input**:

```
Matrix A = [[1, 2], [3, 4]]
Matrix B = [[5, 6], [7, 8]]
```

**Output**:

```
Result = [[6, 8], [10, 12]]
```

---

**Problem**: Subtract the second matrix from the first matrix (both 2x2).

**Input**:

```
Matrix A = [[10, 20], [30, 40]]
Matrix B = [[1, 2], [3, 4]]
```

**Output**:

```
Result = [[9, 18], [27, 36]]
```
